# Java Distributed Chat System

Este projeto consiste em um Sistema de Chat Distribuído desenvolvido em Java, integrando aplicação desktop, aplicação web e comunicação multicast, como parte do estudo prático de Sistemas Distribuídos.
O sistema simula características reais de aplicações modernas de mensagens, incluindo sincronização entre múltiplos clientes, atualização em tempo real e distribuição de publicidade via multicast.
